# Noval Ramadhani Portfolio Website

Portfolio personal untuk role IT Support & Network Engineer.

## Isi Website

- Profil singkat
- Statistik pengalaman
- Skill teknis
- Project cards tanpa foto before-after
- Timeline pengalaman kerja
- Pendidikan
- Kontak

## Cara Deploy ke GitHub Pages

1. Buat repository baru, misalnya `noval-portfolio`.
2. Upload semua file dan folder di dalam paket ini.
3. Masuk ke **Settings** repository.
4. Buka **Pages**.
5. Pada **Build and deployment**, pilih:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. Tunggu 1-3 menit sampai link GitHub Pages muncul.

## Catatan Editing

- Ubah teks di `index.html`.
- Ubah tampilan di `style.css`.
- Ubah interaksi kecil di `script.js`.
- Foto profil ada di `assets/profile.jpg`.
